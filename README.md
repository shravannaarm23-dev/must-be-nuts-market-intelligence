# Must Be Nuts — Market Intelligence Brief

This project is a strategic market intelligence analysis of Must Be Nuts, a premium healthy snacking brand in India.

## Objective

To identify growth opportunities across distribution, digital presence, and competitive positioning.

## Key Insights

* Significant **distribution gap in Q-commerce** (Blinkit, Instamart)
* Product-market-channel fit: **33g packs ideal for impulse buying**
* Strong **content engagement** but weak reach strategy
* Low **social proof (reviews)** impacting conversion at premium pricing

## What This Covers

* Channel gap analysis
* Competitor benchmarking (Happilo, Farmley, True Elements, Nutty Gritties)
* Social media performance insights
* Actionable recommendations
